# Intro to Philosophy
# External World Skepticism

External World Skepticism:
    The claim that it is impossible to truly know if there is a world outside ones mind.

Brain-in-a-vat-Hypothesis:
    You are now a BIV being full of vivid experiences by a supercomputer.

## Argument for EWS

    ES1: If I know I have hands then I know I'm not a BIV.

    ES2: I don't know I'm not a BIV.

    ES3: Therefore, I don't have hands.

### Defense of ES1

        Closure:
            If I know that p entails q, then I know that p only if I know that q.

        Mike & Ike Case, how it supports Closure:
            A case of two near identical things which demonstrate how closure requires the recognition of the second possibility for the first one to be sound.

### Defense of ES2

        2.1: I know that I'm not a BIV only if I have evidence that rule out the possibility that I'm not a BVI.

        2.1: I do not have evidence that rules out the possibility that I'm not a BIV
        
        2.3: Therefore, I don't know I'm not a BIV.
