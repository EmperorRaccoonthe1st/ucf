# Intro to Philosophy 
Freedom and Determinism

## Freedom

    A necessary for freedom:

        The Ability to Do Otherwise Condition:
          Person x does action A freely at t only if x could have done otherwise than A.

## Determinism 

    State of the World (SOW) a time t = a complete list of

        - (i) Every physical object that exists at t
          and
        - (ii) All of their properties at t.

    Determinism:
        The state of the world at any given time t, and the laws of nature necessitate or guarantee a single unique future state of the world at the next time t_2.

    Determinism vs Fatalism:
    
        Fatalism:
           Determinism is true and there was only one possible first state of the world and only one possible set of natural laws.
